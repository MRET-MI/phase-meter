# CP-700M コマンド対応表（本 GUI で使用）

CP-700M の PC ダイレクト通信制御モード（マニュアル §5.2.B.III）。
すべて ASCII、行終端は **CR+LF**。軸トークンは `1`/`2`/`3`（全軸 `W`）。
COMM RES=OFF（初期値）では駆動/設定系コマンドに応答は返らない。

## GUI 操作 → 送信コマンド

| GUI 操作 | 送信コマンド | 備考 |
|---|---|---|
| 接続テスト / 接続直後 | `V:` | バージョン照会（応答あり・疎通確認） |
| 位置・状態ポーリング（300ms） | `Q:` | `±c1,±c2,±c3,ACK1,ACK2,ACK3`（座標[pulse]＋リミット＋受付状態） |
| ジョグ ＋/− 押下 | `D:<ax>S<pps>F<pps>R<ms>` → `J:<ax>+/-` → `G:` | 自起動速度=D の初速で連続運転 |
| ジョグ 離す | `L:<ax>` | 減速停止 |
| 相対移動 | `D:<ax>S<start>F<move>R<ms>` → `M:<ax>±P<pulses>` → `G:` | 方向は符号、パルスは絶対値 |
| 絶対移動 | `D:<ax>S<start>F<move>R<ms>` → `A:<ax>±P<pulses>` → `G:` | 目標座標へ移動 |
| 原点 | `H:<ax>` | 機械原点復帰（方向はパラメータ既定） |
| 0クリア | `R:<ax>` | 現在座標を 0 に |
| 励磁 ON/OFF | `C:<ax>1` / `C:<ax>0` | モータ励磁 |
| 停止 | `L:<ax>` | 指定軸を減速停止 |
| 全軸即停止 | `L:E` | 非常停止相当（全軸即停止） |
| 直接コマンド送信 | 入力文字列そのまま | 応答を読む（`expect_response=True`） |

## 単位換算（`core/config.py`）

```
mm_per_pulse = lead_mm / (500 × divide)          （マニュアル §6.1.A）
pulses = round(mm / mm_per_pulse)
pps    = max(1, round((mm/s) / mm_per_pulse))
```

## Q: 応答の解釈（`core/controller.py: parse_status`）

```
"+  100,+  200,+    0,K,7,R"
 └─座標1  └─座標2  └─座標3  │ │ └ ACK3: R=全受付可 / B=一部 / I=インターロック中
                            │ └── ACK2: リミットビット（0..7）
                            └──── ACK1: K=正常受付 / X=エラー
```
