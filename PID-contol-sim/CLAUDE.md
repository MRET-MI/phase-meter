# ルール

## 操作範囲
- 操作対象は必ずこのフォルダ（カレントディレクトリ）内のみ
- 上位フォルダには絶対に触らない
- デスクトップやドキュメントフォルダは操作禁止

## ファイル操作
- ファイルを削除する前に必ず確認を求めること

## 言語
- 返答は必ず日本語で行うこと

## 作業スタイル
- 作業前に「何をするか」を説明してから実行すること
- 一度に大量の変更をしないこと
- 実行は別途コマンドプロンプトを立ち上げておりそちらで確認する。

# GUI
- GUI作成時はPythonのPyside6を使うこと
- UIはpyside-designerで編集できるようにすること。

# ライブラリ
python-controlモジュールを使うこと

# 検討条件
・ペルチェは387006916
・TECドライバ ADN8834
   TECドライバのIN1Nにサーミスタの抵抗変化に伴う電圧を入力し、OUT1を16ビットでADCする。16bitで電圧値を指定するDACを使いTemperature Set電圧をIN2Pに入力しペルチェに与える電圧を制御する
・サーミスタのB定数は3380
・ADCした値を温度に変換せず、そのままPID制御の値として使う。
・目標温度は25℃に設定する。
・外乱としてレーザーの発熱を考える。最大2Wの発熱がある。
